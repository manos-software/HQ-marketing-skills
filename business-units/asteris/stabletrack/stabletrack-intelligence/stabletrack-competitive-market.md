# StableTrack — Competitive Landscape
> Reference file. Load for comparison content, sales enablement, or positioning against competitors.

**Rule:** Never disparage competitors by name in customer-facing content. Focus on category weaknesses: "tools built for clinics," "systems that require internet."

---

## Direct equine-specific competitors

**Cassadol** (by Business Infusions)
- Target: Solo equine vets (1–2 vets)
- Price: $59/mo per user + add-ons
- Strengths: Simple setup (20 min), affordable, cloud-based, QuickBooks integration, racetrack split billing
- Weaknesses: No inventory management (except controlled drugs), no calendar/scheduling, no AI documentation, offline is paid add-on, limited integrations (CardConnect only)
- Vulnerability: Records-and-billing tool, not a full PMS. Explicitly states it "does not do everything a full-scale practice management software program does."

**HVMS** (by Business Infusions — same parent as Cassadol)
- Target: Medium-large equine practices (3+ vets)
- Price: $385/mo site license + $185/mo per vet + add-ons. $6K setup fee.
- Strengths: Comprehensive features, advanced inventory, vet compensation models, whiteboard/treatment plans, detailed reporting, multi-location. Claims 5–20% profitability increases.
- Weaknesses: Steep learning curve, frequent glitches reported by users, slow support, syncing issues for ambulatory use, expensive for small practices. 3.7/5 on GetApp (61% positive).
- Vulnerability: Built for large hospital-based practices. Overpowered/expensive for 1–4 vet ambulatory practices.

**StableTrack's position:** Fills the gap between Cassadol (too basic) and HVMS (too complex/expensive).

**Covetrus Impromed Equine**
- Part of Covetrus ecosystem (Pulse PMS). Legacy system, enterprise-oriented.

## General vet PMS with equine features

**DaySmart Vet** — Cloud-based, 40+ integrations, "Daisy Voice" dictation (transcription only, not SOAP generation). 4.6/5. Not truly offline-first. Clinic-first design with equine added on.

**ezyVet** — Modern cloud platform, syndicate billing, "ezyVet Go" for field. 4.6/5. Built for clinic workflows. Equine is a feature set, not core design. Requires internet.

**Provet Cloud** — Customizable workflows, multi-patient appointments. Not equine-first.

**Vetport** — Group billing, mobile usability. Generic design.

---

## Adjacent / emerging

**ThoroVet** — Cloud, iPad-focused, QuickBooks, equine-specific, offline capable. Worth monitoring.

**Shepherd Veterinary Software** — Modern cloud PMS, "workflow-first" philosophy, record-to-invoice automation. 4.8/5 on GetApp. Companion animal but philosophically aligned. Could expand to equine.

**AI scribe tools (CoVet, TalkingVet)** — Not PMS competitors but could reduce perceived value of StableTrack's native AI docs if layered on top of existing PMS.

## StableTrack's moat
