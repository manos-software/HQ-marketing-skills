# ICP — StableTrack

## Primary ICP
Independent equine ambulatory veterinarian

## Firmographics
- 1–4 veterinarians
- Mobile / ambulatory practice
- Rural or semi-rural geography
- Limited admin support

## Daily workflow
- Travel between barns/farms
- Perform exams and treatments on-site
- Document findings manually or later
- Invoice after visits (often delayed)

## Core problems
- Services performed but not billed
- Invoices created hours or days later
- Details forgotten or lost
- Admin work pushed to evenings

## Buying triggers
- Realizing revenue is being lost
- Frustration with current software
- Practice growth increasing complexity
- Desire to reduce admin time

## Objections
- "Switching systems is painful"
- "I don’t have time to learn new software"
- "This is too expensive"
- "My current system works well enough"

## Buying motivations
- Increase revenue without increasing workload
- Simplify workflow in the field
- Reduce after-hours admin work
- Use a system built specifically for equine practice
