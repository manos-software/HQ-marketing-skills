# Positioning — StableTrack
> version: 1.0.0

## Category
Equine practice management software (PMS)

## Positioning statement
The first AI-native PMS built specifically for equine veterinarians working in the field.

## Primary angle
Revenue recovery, not efficiency.

## Core narrative
Equine vets lose revenue through small, compounding failures:
- services not recorded
- fees not applied
- invoices delayed or forgotten

StableTrack closes that loop in real time.

## Competitive positioning

### vs Cassadol
- Cassadol: basic equine tool, limited features
- StableTrack: full system (scheduling, billing, inventory, AI, offline)

### vs general vet software
- Built for clinics, not field work
- Require internet
- Do not support equine workflows

StableTrack is purpose-built for ambulatory equine vets.

## Key differentiators to reinforce
- Offline-first (critical)
- AI documentation
- Billing integrated into workflow
- Multi-owner horse management

## Proof points
- Designed specifically for equine workflows
- Built on Asteris infrastructure
- Native imaging integration
